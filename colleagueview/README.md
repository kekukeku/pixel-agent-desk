# Colleague View Retrospectives

This directory stores cross-agent retrospective notes created when the human operator says `盤點此次任務`.

All three agents may read and write Markdown files in this directory:

- Codex: `codex_view_NNN.md`
- Antigravity: `antigravity_view_NNN.md`
- Grok Build: `grok_view_NNN.md`

Each file should evaluate the other two colleagues for the latest completed task, covering concrete strengths, constructive improvement suggestions, and an overall impression.
